# Retapis 2
A JavaScript graphics engine.

## How to start?
  Just copy the .js file (retapis2.min.js or retapis2.js) into your project directory.
  
  Include in HTML using `script`:
  
  `<script src='scripts/retapis2.min.js'></script>`
